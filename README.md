# docker-platex
The docker image for building tex to pdf.

# useage
download `tex2pdf.sh` on directory there is target tex file, and run `./tex2pdf.sh [SCRIPT_NAME] [tex file]`.

## SCRIPT_NAME
below scripts are available in container.

* build
* build-bibtex
* build-pbibtex
* build-biber
